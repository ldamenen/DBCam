        About this demo version

        This is an early demo to try out the idea — not the finished product. It deliberately does not provide:

        • Encryption at rest — no Secure-Enclave / Keystore key storage; the raw blob is held in memory.
        • Always-on / background recording — browsers suspend hidden tabs; the screen wake lock is best-effort.
        • Cryptographic sealing / split-key authorization — the flow is demonstrated, but "authorization" is a UI gate and playback is only clamped, not cryptographically sealed.
        • Jurisdiction enforcement — the region rules are demo values (not legal advice) and there is no automatic location detection yet; you choose the region in Settings, and until you do the strictest rules apply.

        These guarantees are native-only (iOS Vision/CryptoKit, Android ML Kit/Keystore). The animal deterrent may help avoid, definitely helps document — it is not a safety guarantee.
